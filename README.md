## Local Development 

MacOS

1. Create and activate virtual environment:
```bash
python3 -m venv venv
source venv/bin/activate
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the app:
```bash
flask run
```

4. Open http://localhost:5000

Windows

1. Create and activate virtual environment:
```bash
python -m venv venv
venv/Scripts/Activate
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the app:
```bash
app.py
```

4. Open http://localhost:8000
